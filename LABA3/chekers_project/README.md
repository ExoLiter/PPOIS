# Checkers - Лабораторная работа №3

Проект представляет собой реализацию игры "Английские шашки" на языке Python с использованием библиотеки `pygame`. Выполнен согласно требованиям событийно-ориентированного программирования.

## Особенности реализации
- **Режим игры**: Hotseat (Два игрока за одним экраном).
- **Переворот доски**: После каждого хода доска автоматически переворачивается (отрисовывается с точки зрения активного игрока), согласно требованию ТЗ.
- **Анимации**: Плавное перемещение шашек при ходе с использованием линейной интерполяции координат.
- **Рекорды**: Подсчет количества побед по именам. Сохранение и вывод Топ-10 лучших игроков.

## Архитектура и качество кода
Код написан с соблюдением строгих правил "Чистого кода":
1. Отсутствие хардкода: Все числовые значения, пути и настройки вынесены во внешний `config.json`.
2. Малый размер функций: Каждая функция строго выполняет одну задачу и занимает менее 20 строк.
3. Низкая вложенность: Не более 3 уровней вложенности конструкций if/for/while.
4. Документирование: Каждая функция и класс содержат docstring-описание на русском языке.

## Установка и запуск
1. Установите зависимости: `pip install pygame`
2. Создайте папку `assets/` в корне проекта и добавьте в неё медиафайлы: `music.mp3`, `click.wav`, `move.wav`, `win.wav`.
3. Запустите игру командой: `python main.py`
