# Лабораторная работа №1 (вариант 55)

## Тема
Модель готовки яичницы с интерфейсом командной строки (CLI).

## Предметная область
Процесс приготовления блюда с обязательными сущностями:
- яйца
- сковорода
- масло
- приправы
- плита
- лопатка

## Реализованные операции
1. Разбить яйца (с выбором количества, накопительно, максимум 10)
2. Включить электроплиту (выбор мощности 1-6 и времени готовки в минутах)
3. Добавить масло (сливочное или подсолнечное)
4. Обжарить яйца (с выбором, сколько из разбитых яиц обжарить)
5. Добавить приправы (соль, перец, паприка, универсальная)
6. Перемешать и подать блюдо
0. Выход

## Соотношение мощности и времени
- Мощность 1: 15-20 минут
- Мощность 2: 11-14 минут
- Мощность 3: 7-10 минут
- Мощность 4: 5-6 минут
- Мощность 5: 3-4 минут
- Мощность 6: 1-2 минут

Если время меньше диапазона, при подаче выводится: яйца получились сырые.  
Если время больше диапазона, при подаче выводится: яйца сгорели.  
В обоих случаях указывается правильный диапазон времени для выбранной мощности.

После подачи выводится итог:
- сколько яиц обжарено
- сколько яиц осталось сырых (необжаренных)

## Выполненные требования
- Объектно-ориентированная модель предметной области.
- CLI-меню выбора операций по номеру.
- Аннотации типов.
- Обработка ошибочных ситуаций через исключения.
- Unit-тесты.

## Структура проекта
- `main.py` — точка входа (только запуск CLI).
- `cooking/cli.py` — слой интерфейса командной строки.
- `cooking/process.py` — прикладная логика (оркестрация операций).
- `cooking/entities.py` — доменные сущности (яйца, плита, сковорода, масло, приправы, лопатка).
- `cooking/exceptions.py` — доменные исключения.
- `tests/test_cooking.py` — unit-тесты.

## Запуск программы
```bash
py -3 main.py
```

## Запуск тестов
```bash
py -3 -m unittest discover -s tests -v
```
